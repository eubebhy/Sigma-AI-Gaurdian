# Quy ước file keyword

- Mỗi dòng là một nhóm từ khóa liên quan.
- Các keyword trong cùng một dòng được ngăn cách bằng dấu `|`.
- Tên file phải khớp đúng với nhóm nội dung tương ứng, ví dụ `game.txt`, `pornography.txt`, `gore.txt`.
- Tránh keyword quá ngắn hoặc quá phổ biến để hạn chế false positive.

## Ví dụ

```text
cs go | counter strike
minecraft
```
